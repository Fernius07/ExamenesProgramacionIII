# Posible examen extraordinario 2026 - Programación III

Este examen propone 3 tareas prácticas sobre:

1. **Recursividad**
2. **JTable (modelo de tabla)**
3. **Threads + Swing**

## Estructura

- `src/es/deusto/prog3/examen2026extra/...` contiene los **archivos fuente para el alumnado** con los `TODO`.
- La carpeta gemela `../2026_extraordinaria_solucion` incluye una posible resolución completa.

## Tareas

### Tarea 1 - Recursividad (4 puntos)
Completa el método `sumarMinutosRec` para calcular de forma recursiva el total de minutos de actividades que cumplan tipo e intensidad.

Archivo: `recursividad/MainRecursividad.java`

### Tarea 2 - JTable (3 puntos)
Completa el modelo `TablaActividadesModel` para que la tabla muestre correctamente los datos y permita editar la columna de minutos.

Archivo: `swing/TablaActividadesModel.java`

### Tarea 3 - Threads + Swing (3 puntos)
Implementa el inicio y parada de hilos en `MainSemaforo` para que cada panel cambie de color con periodo aleatorio, respetando EDT para la actualización visual.

Archivo: `threads/MainSemaforo.java`
